# Testes 🧪

## 🛠️ Backend

Para executar os testes do backend, basta executar o comando abaixo:

- Abrir a pasta `backend` ou rodar o comando abaixo:

<!-- termynal -->

```console
$ cd backend
```

- Executar o PyTest com o comando abaixo:

<!-- termynal -->

```console
$ poetry run task test
```

> 💡 Você pode conferir a cobertura de testes acessando o arquivo gerado `htmlcov/index.html` no seu navegador.
